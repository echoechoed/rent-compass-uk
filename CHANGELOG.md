# Changelog

All notable changes to Rent Compass UK are recorded here. The project follows [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-07-23

### Added

- Initial public beta of the UK rental search skill.
- Separate guidance for England, Scotland, Wales, and Northern Ireland.
- Renter intake, area discovery, multi-destination commute handling, live search, listing verification, cost comparison, ranking, and pre-commitment checks.
- Git-backed plugin marketplace packaging.
- English and Simplified Chinese documentation.
- Human validation checklist and scenario suite.
